# ToDo List Project

Here you can write your to dos, you can check it when you complete it, also you can remove.

## Clone This Project
```
- git clone `$ https://github.com/elyor-doniyorov/TODO-LIST.git`
- cd TODO-LIST
- Type `npm install`.
- Now type `npm start`, YAY you can now see the project.
```
